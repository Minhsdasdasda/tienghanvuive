# Tiếng Hàn vui vẻ cùng cô Lan

Web tĩnh để luyện tiếng Hàn TOPIK 1-3, chạy trực tiếp bằng GitHub Pages.

Kho hiện có 1000 câu. Khi chọn TOPIK 1, đề chỉ lấy câu cấp 1; chọn TOPIK 2 thì lấy cấp 1 và 2; chọn TOPIK 3 thì lấy cấp 1, 2 và 3.

## Cách chạy

Mở `index.html` trong trình duyệt, hoặc bật GitHub Pages cho repository này.

## Cách thay ảnh cô Lan

Thêm ảnh thật vào thư mục `assets` và đặt tên là `co-lan.jpg`. Trang sẽ tự dùng ảnh đó; nếu chưa có thì dùng ảnh minh họa `co-lan.svg`.

## Ghi chú về audio

Phần nghe dùng Web Speech API của trình duyệt để đọc tiếng Hàn, không dùng file audio lấy từ nguồn bên ngoài. Chất lượng giọng đọc phụ thuộc vào trình duyệt và hệ điều hành.
